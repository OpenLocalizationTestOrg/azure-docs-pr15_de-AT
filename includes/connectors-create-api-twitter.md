### <a name="prerequisites"></a>Erforderliche Komponenten
- Twitter-Konto 

Bevor Ihr Twitter-Konto in eine Logik-app verwenden können, müssen Sie Logik app Verbindung mit Twitter-Konto autorisieren. Glücklicherweise können Sie problemlos in der Logik app Azure-Portal dazu. 

Hier sind die Schritte ermächtigen Ihre Logik app Verbindung mit Twitter-Konto:

1. Zum Erstellen einer Verbindung mit Twitter, Logik app Designer **Anzeigen von Microsoft verwalteten APIs** in der Dropdown-Liste Wählen Sie *Twitter* in das Suchfeld eingeben. Wählen Sie den oder die Aktion, die Sie mögen verwenden:  
  ![Twitter Verbindung Bild 0](./media/connectors-create-api-twitter/twitter-0.png)
2. Wenn Sie Verbindung mit Twitter vor erstellt haben, abrufen Sie aufgefordert der Twitter Anmeldeinformationen. Diese Anmeldeinformationen verwendet Ihre Anwendung Logik Verbindung autorisiert und Zugriff auf Ihr Twitter-Konto-Daten:  
  ![Twitter Verbindung Bild 1](./media/connectors-create-api-twitter/twitter-1.png)  
3. Geben Sie Ihre Twitter-Benutzernamen und Kennwort ermächtigen Logic app:  
  ![Twitter Verbindung Bild 2](./media/connectors-create-api-twitter/twitter-2.png)  
4. Die Autorisierung zu bestätigen:  
  ![Twitter Verbindung Bild 3](./media/connectors-create-api-twitter/twitter-3.png)  
6. Beachten Sie die Verbindung eingerichtet wurde und Sie können die Schritte in Ihrer Anwendung Logik fortsetzen:  
  ![Twitter Verbindung Bild 4](./media/connectors-create-api-twitter/twitter-4.png)
