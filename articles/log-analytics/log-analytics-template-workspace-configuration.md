# <a name="manage-log-analytics-using-azure-resource-manager-templates"></a>Verwalten Sie Protokollanalyse Azure Ressourcenmanager Vorlagen

[Azure Ressourcenmanager Vorlagen] verwenden (... / azure-resource-manager/resource-group-authoring-templates.md) erstellen und Konfigurieren der Protokollanalyse Arbeitsbereiche. Beispiele für Aufgaben mit Vorlagen ausführen:

+ Erstellen eines Arbeitsbereichs
+ Hinzufügen einer Lösung
+ Erstellen Sie gespeicherte Suchen
+ Erstellen einer Computergruppe
+ Aktivieren Sie Auflistung von IIS-Protokollen auf Computern mit installiertem Windows-agent
+ Sammeln von Leistungsindikatoren von Linux und Windows
+ Sammeln von Ereignissen von Syslog auf Linux-Computern 
+ Ereignisse von Windows-Ereignisprotokolle erfassen
+ Benutzerdefinierten Ereignisprotokolle erfassen
+ Den Agent Analytics Azure virtuellen Computer hinzufügen
+ Konfigurieren der Protokollanalyse mit Indexdaten mithilfe von Azure diagnostics


Dieser Artikel enthält eine Vorlage Beispiele für die Teil der Konfiguration, die aus Vorlagen ausführen können.

## <a name="create-and-configure-a-log-analytics-workspace"></a>Erstellen und konfigurieren ein Protokoll Analytics-Arbeitsbereich

Die folgende Vorlage veranschaulicht wie:

1.  Erstellen eines Arbeitsbereichs
2.  Projektmappen Arbeitsbereich hinzufügen
3.  Erstellen Sie gespeicherte Suchen
4.  Erstellen einer Computergruppe
5.  Aktivieren Sie Auflistung von IIS-Protokollen auf Computern mit installiertem Windows-agent
6.  Logischer Datenträger Leistungsindikatoren von Linux-Computer sammeln (% Inodes verwendet; MB frei; % Belegt; / Sek. übertragen; Lesevorgänge/s; Schreibvorgänge/s)
7.  Syslog-Ereignisse von Linux-Computern erfassen
8.  Erfassen Sie Fehler und Warnung Ereignisse in das Anwendungsereignisprotokoll von Windows-Computern
9.  Sammeln Sie Verfügbare MB Arbeitsspeicher-Leistungsindikator von Windows-Computern
10. Sammeln Sie ein benutzerdefiniertes Protokoll 
11. Sammeln Sie IIS-Protokolle und Windows-Ereignisprotokolle geschrieben von Azure Diagnostics ein Speicherkonto

### <a name="deploying-the-sample-template"></a>Vorlage bereitstellen

Vorlage bereitstellen:

1. Beispielsweise speichern Sie angefügte Beispiel in einer Datei`azuredeploy.json` 
2. Bearbeiten Sie die Vorlage, um die Konfiguration haben
3. Verwenden Sie PowerShell oder die Befehlszeile, um die Vorlage bereitstellen

#### <a name="powershell"></a>PowerShell

`New-AzureRmResourceGroupDeployment -Name <deployment-name> -ResourceGroupName <resource-group-name> -TemplateFile azuredeploy.json`

#### <a name="command-line"></a>Befehlszeile

```
azure config mode arm
azure group deployment create <my-resource-group> <my-deployment-name> --TemplateFile azuredeploy.json
```
