# <a name="discover-competitions-in-the-cortana-intelligence-gallery"></a>Entdecken Sie Wettbewerbe im Cortana Intelligence Katalog

[AZURE.INCLUDE [machine-learning-gallery-item-selector](../../includes/machine-learning-gallery-item-selector.md)]

## <a name="cortana-intelligence-competitions"></a>Cortana Intelligence Auswahlverfahren

**[Wettbewerbe](https://gallery.cortanaintelligence.com/competitions)** bieten eine attraktive Chance zu mit datenwissenschaftler lösen komplexer Probleme mit Cortana Intelligence Suite.

## <a name="discover"></a>Entdecken

  Durchsuchen für Wettbewerbe in der Galerie [Galerie](http://gallery.cortanaintelligence.com)öffnen, auf Maus **mehr** am oberen Rand der Galerie-Homepage und **Auswahlverfahren**wählen.

![Gallery-Startseite Auswahlverfahren auswählen](media/machine-learning-gallery-competitions/select-competitions-in-gallery.png)

 Die **[Auswahlverfahren](https://gallery.cortanaintelligence.com/competitions)** 
 Seite zeigt eine Liste der beliebtesten Wettbewerbe.
Klicken Sie auf **Alle** , um alle Auswahlverfahren anzuzeigen.
Auf dieser Seite finden Sie die Wettbewerbe in der Galerie oder Suche können Sie Filter Kriterien auf der linken Seite der Seite und der Eingabe von Suchbegriffen oben.

 Klicken Sie auf Wettbewerb Detailseite für den Wettbewerb zu öffnen und Weitere Informationen. Auf dieser Seite können Sie Kommentare, Feedback geben und Fragen durch den Kommentaren. Sie können auch mit Freunden oder Kollegen mit der Freigabe von LinkedIn oder Twitter freigeben. Sie können auch eine Verknüpfung mit der Konkurrenz Einladen anderer Benutzer zum Anzeigen der Seite e-Mail.

![Dieses Element mit Freunden teilen](media\machine-learning-gallery-how-to-use-contribute-publish\share-links.png)

![Eigene Kommentare hinzufügen](media\machine-learning-gallery-how-to-use-contribute-publish\comments.png)

## <a name="enter-a-competition"></a>Geben Sie einen Wettbewerb

**Ist der Wettbewerb geöffnet, wird der Status auf der Seite aktiviert.** Zum Gewinnspiel auf **Wettbewerb Geben Sie** erhalten Anweisungen auf die benötigten Ressourcen und die Schritte zu.

![Wettbewerb ist abgeschlossen](media\machine-learning-gallery-competitions\open-competition.png)

Ist der Wettbewerb mehr, den Status auf der Detailseite werden **abgeschlossen** und Word **beendet**der **Wettbewerb geben** Link ersetzt werden.

![Wettbewerb ist abgeschlossen](media\machine-learning-gallery-competitions\completed-competition.png)
