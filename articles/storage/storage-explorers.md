# <a name="azure-storage-client-tools"></a>Azure-Speicher-Clienttools

Häufig möchten Benutzer von Azure Storage View/ihre Daten mithilfe einer Azure Storage-Clienttool interagieren können. In den folgenden Tabellen Listen wir eine Reihe von Tools, mit denen Sie dazu. Wir haben ein "X" in jedem Block stellt die Möglichkeit, auflisten oder die Datenabstraktion zugreifen. Die Tabelle zeigt auch wenn Tools frei oder nicht. "Test" zeigt an, dass eine kostenlose Testversion ist die Vollversion ist nicht kostenlos. "Y/N" gibt an, dass eine Version verfügbar, eine andere Version erhältlich ist.

Wir haben lediglich einen Snapshot der Azure-Speicher Client Werkzeuge bereitgestellt. Diese Tools können weiterhin wachsen Funktionalität. Wenn Korrekturen oder Updates vorhanden sind, geben Sie einen Kommentar um uns mitzuteilen. Dies gilt auch, wenn Sie Tools, die kennen-wir fügen sie gerne.
